# 基本界面
 
## Workspace 工作台

工作台包括应用、模块和数据集。用户可以查看每个分类下自己的开发项目，并通过顶部下拉菜单通过权限筛选项目。

点击“新建项目”按钮创建项目，点击“Notebook”进入工作区，Notebook使用参考开发指南。
